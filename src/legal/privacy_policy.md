## 2. Какие данные собирает Сервис

2.1. **Идентификационные данные Telegram:** Telegram ID, username (никнейм), имя в Telegram, код языка интерфейса. Эти данные передаются автоматически Telegram-платформой при взаимодействии с ботом.

2.2. **Данные для генерации контента:** имя ребёнка, возраст ребёнка, предпочтения по теме сказки, любимый персонаж. Указываются Пользователем добровольно при создании сказки.

2.3. **Платёжная информация:** email Пользователя (запрашивается ЮKassa для отправки чека по 54-ФЗ), идентификатор платёжного метода (`payment_method_id`) для последующих рекуррентных списаний. Реквизиты банковских карт **не сохраняются** на стороне Оператора — обработка карт производится платёжной системой АО «ЮMoney» (ЮKassa) по стандарту PCI DSS.

2.4. **История использования:** созданные сказки (текст, аудио, обложка), оплаты, дата регистрации, дата последней активности, источник перехода (UTM-метка).

2.5. **Технические данные:** логи запросов к API, время и тип операций. Хранятся 30 суток, используются для отладки и предотвращения злоупотреблений.

3.3. Информирование Пользователя об операциях с подпиской через сам Telegram-бот.

3.5. Обеспечение безопасности Сервиса и предотвращение злоупотреблений (в частности — повторного бесплатного использования через `/delete_me`).

## 4. С кем Оператор делится данными

Персональные данные передаются третьим лицам **только в объёме, необходимом для оказания услуги**, на основании договоров о неразглашении и обработке персональных данных:

— **АО «ЮMoney» (ЮKassa)** — для проведения платежей и формирования чеков 54-ФЗ. Передаётся: сумма платежа, email Пользователя, описание услуги.

— **Google LLC (Gemini API)** — для генерации текста сказки. Передаётся: имя ребёнка, возраст, тема, любимый герой. Согласно политике Google для платных API, передаваемые промпты не используются для обучения моделей.

— **ElevenLabs Inc. (TTS API)** — для синтеза речи. Передаётся: текст сказки (содержащий имя ребёнка).

— **fal.ai** — для генерации иллюстраций. Передаётся: описание сцены (без личных данных).

— **OOO «Сбер AI» (FusionBrain / Kandinsky)** — резервный провайдер иллюстраций. Условия аналогичны fal.ai.

— **АО «Тинькофф Банк» / СПАО «Сбербанк России»** — для проведения партнёрских выплат блогерам, если такие выплаты возникают.

Оператор **не передаёт** персональные данные третьим лицам в коммерческих целях, не продаёт базы и не использует их для таргетированной рекламы за пределами Сервиса.

## 5. Сроки хранения

5.1. Данные Telegram-аккаунта и история использования хранятся в течение всего срока использования Сервиса.

5.2. После запроса на удаление (команда `/delete_me` в боте или письмо на denis.aivatrov@yandex.ru) персональные данные удаляются в течение 24 часов:

— Полное удаление учётной записи доступно по запросу через `denis.aivatrov@yandex.ru`.

— Команда `/delete_me` в боте выполняет частичное удаление (стираются: имя ребёнка, никнейм, способ оплаты, реферальная ссылка). Запись о факте регистрации Telegram-аккаунта сохраняется в обезличенном виде в целях предотвращения злоупотреблений бесплатным пробным периодом.

5.3. Сведения о совершённых платежах хранятся 5 лет согласно требованиям налогового законодательства РФ.
