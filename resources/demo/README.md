# Витринный образец сказки

Эта папка — fallback-источник для витрины («🌟 Посмотреть образец сказки»
в главном меню бота). Используется, если в `cache/demo/` (persistent
volume) админ ещё не положил живой образец через `/save_as_demo`.

## Ожидаемые файлы

- `story.txt` — текст сказки. Первая строка — название, далее пустая
  строка, далее сам текст.
- `cover.jpg` — обложка (opening illustration).
- `climax.jpg` — кульминационная иллюстрация (опционально).
- `ending.jpg` — финальная иллюстрация (опционально).
- `book.pdf` — собранная PDF-книжка.

## Как обновлять

Рекомендуемый путь — через бот в проде:

```
/save_as_demo <story_id>
```

Эта команда копирует файлы из `cache/images/` и `cache/pdfs/` в
`cache/demo/`, override'я default из этой папки. Витрина обновляется
без рестарта бота и без коммита в репо.

Если хочется обновить и default-фоллбек (например, после смены
стилистики), сгенерируй на проде, скачай файлы через `docker cp`, и
сделай коммит в эту папку.
