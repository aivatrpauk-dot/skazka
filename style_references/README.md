# Эталонные картинки для Recraft Custom Style

Текущий набор — версия 3 (май 2026, под позиционирование «сказка на ночь»).
Предыдущие наборы:
- `../style_references_old/` — самые первые 5 (часть были AI-генерации).
- `../style_references_ai_backup/` — две AI-картинки из v2, которые мы выкинули
  в v3 (они тащили стиль в «AI-вид с толстым контуром» и оставляли в выводе
  артефакты UI Recraft в виде кружков и цифр).

## Принцип набора

Продукт — персонализированная сказка перед сном для детей 3-6.
Визуал смотрит **родитель** вечером (ребёнок засыпает с закрытыми
глазами), и появляется в PDF — где это уже семейный артефакт для
дневного просмотра, печати, шеринга.

Целевая эстетика: **«сказка на ночь»** — тёплый вечерний свет,
антропоморфные звери в уютных интерьерах, плотный обжитый сказочный
мир, премиальная акварель + видимая тушь.

**Жёсткое правило для датасета:** все 5 картинок должны быть **настоящими
сканами книг**, никаких AI-генераций. AI-картинки от другой модели создают
feedback loop и тащат с собой UI-артефакты (кружки, номера, иконки), которые
Recraft принимает за элемент стиля и запекает в каждую генерацию.

## Манифест

### 01 — Антон Ломаев, «Колыбельная для маленького пирата»
**Что:** папа-пират читает сказку маленькому пирату перед сном.
**Зачем:** прямое попадание в позиционирование. Тонкая тушь + плотная акварель,
премиальная книжная иллюстрация. Источник — фото книжного разворота.

### 02 — Беатрикс Поттер, Mrs. Tiggy-Winkle на кухне
**Что:** ёжик-прачка в платье у полок с посудой, с бельём в руках.
**Зачем:** канонический «антропоморфный герой в тёплом домашнем интерьере».
Источник — Project Gutenberg, public domain.

### 03 — Беатрикс Поттер, Two Bad Mice
**Что:** Том Тамб и Ханка Манка за обеденным столом в кукольном домике —
ветчина, фрукты, омары, розовые обои.
**Зачем:** ещё один антропоморфный интерьер, тёплая палитра (розовый, бежевый),
полный fund of detail. Источник — Project Gutenberg, public domain.

### 04 — Геннадий Спирин, мальчик с книгой в кресле
**Что:** мальчик в красном кресле поглощён большой книгой.
**Зачем:** премиальная российская акварельная иллюстрация. Тёплая палитра
(красные чулки, тёмная мебель), читаемый герой в спокойной позе чтения.

### 05 — Беатрикс Поттер, Squirrel Nutkin и Old Brown
**Что:** Старый Браун-сова в дупле дерева, белки вокруг с дарами.
**Зачем:** многоплановая сцена, тёплая бежево-зелёная палитра, лесной интерьер
(дупло дерева), 5+ героев. Балансирует «холодные» сцены интерьеров тёплой
природной сценой. Источник — Project Gutenberg, public domain.

## Что НЕ годится в этот датасет

- **AI-генерации любого происхождения.** Главный урок v2: даже красивая
  AI-картинка приносит с собой свою «AI-эстетику» с резким контуром и
  иногда UI-артефактами. Только сканы книг.
- Сутеев, Чарушин — слишком детские/советские, тянут стиль в «обычная
  кид-книжка», теряем премиум-восприятие.
- Tove Jansson — монохромный скретчборд, конфликтует с цветной акварелью.
- Анимационные стили (Ghibli, Pixar), 3D, аниме, манга, комиксы.
- Мрачные взрослые иллюстраторы (Калиновский, Сендак, поздний Олейников) —
  продукт для bedtime, не для horror.

## Тренировка

```
export RECRAFT_API_KEY=...
python scripts/create_recraft_style.py
```

Скрипт возьмёт все 5 картинок из этой папки, загрузит в Recraft,
вернёт `style_id` вида `12345678-abcd-...`. Этот id прописать в
`.env` как `RECRAFT_STYLE_ID=...` и задеплоить.
