# Эталонные картинки для Recraft Custom Style

Версия 4 (май 2026, окончательное направление — **Brambly Hedge + Ломаев**).

Предыдущие наборы:
- `../style_references_old/` — первые AI-генерации
- `../style_references_ai_backup/` — две AI-картинки v2 (зайчиха, повозка)
- `../style_references_v3_backup/` — муторные Поттер/Спирин v3, ушли из-за меланхолии

## Эволюция через 4 итерации

1. **v1 (исходный):** 5 AI-картинок с UI-артефактами → выводы выглядели как AI с толстым контуром
2. **v2:** заменил на «классические» — Сутеев/Чарушин/Поттер → слишком детская советская эстетика
3. **v3 (Поттер/Спирин/Ломаев):** премиум, но **меланхолия** — графитный карандаш, сепия, опущенные глаза, виньетки
4. **v4 (текущая, Brambly Hedge + Ломаев + Поттер Two Bad Mice):** яркая, тёплая, плотная — то что и нужно для «сказка на ночь»

## Принцип v4

Продукт — персонализированная сказка перед сном для детей 3-6. Визуал
смотрит **родитель** вечером + появляется в PDF как семейный артефакт.

Целевая эстетика: **Brambly Hedge** — детальный обжитый мир, где живут
антропоморфные звери в человеческой одежде. Тёплые насыщенные цвета,
лето и тёплая осень, никакой зимы и сепии. Множество мелких деталей в кадре.

**Доминирующий автор:** Jill Barklem (Brambly Hedge) — 3 из 5 картинок.
Это даёт модели сильную направленность, а не «среднюю по всем».

## Манифест

### 01 — Антон Ломаев, «Колыбельная для маленького пирата» (без текста)
**Что:** папа-пират читает сыну, рядом дрейфующие сны-облака с китами.
Обрезано чтобы убрать поэтический текст из угла.
**Зачем:** премиальная баррочная акварель + тонкая тушь.
Прямое попадание в тематику продукта (бедтайм-чтение).

### 02 — Беатрикс Поттер, Two Bad Mice
**Что:** Том Тамб и Ханка Манка за обеденным столом в кукольном домике —
ветчина, фрукты, омары, розовые обои.
**Зачем:** **самая яркая Поттер** — насыщенный цвет, антропоморфный
интерьер, тёплая палитра. Не виньетка.

### 03 — Jill Barklem, Palace Kitchen
**Что:** мышиная кухня — стеллажи с банками заготовок, тыквы и яблоки
на потолке, мышь в фартучке мешает тесто.
**Зачем:** **главный референс «богатой плотной среды»**. Это и есть та
самая «зайчиха в дупле»-эстетика, только настоящая (не AI).

### 04 — Jill Barklem, бедтайм мышей (обрезано)
**Что:** мама-мышь укладывает мышат в кровать с лоскутным одеялом,
обои с цветами, плюшевые игрушки на полу.
**Зачем:** **прямое тематическое попадание** — бедтайм. Подпись внизу
обрезана, чтобы стиль не учился генерировать текст.

### 05 — Jill Barklem, мыши под боярышником
**Что:** мыши в чепцах сидят с малышами под цветущим деревом, корзинка
для пикника, бабочки.
**Зачем:** **антипод зимнего крена** — летняя сцена с цветущим деревом
и пикником. Тёплая палитра, плотная среда.

## Жёсткие правила набора

- **Никаких AI-генераций** — feedback loop и UI-артефакты.
- **Никакого видимого текста на картинках** — стиль учится дорисовывать
  заголовки в иллюстрации.
- **Никаких сепий/графитов/виньеток** — это и есть «меланхолия» в выводе.
- **Никаких зимних/холодных сцен** — продукт «лето» по DNA.
- **Антропоморфные звери в человеческой одежде** — обязательное ядро.
