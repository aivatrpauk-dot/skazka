# Эталонные картинки для Recraft Custom Style

Сюда нужно сложить **ровно 5 картинок** под именами `01.jpg` … `05.jpg`
(или `.png` — формат не важен). По ним Recraft натренирует приватный
`style_id`, который потом будет применяться ко всем картинкам в боте.

Под каждый номер описано, какая именно картинка нужна (это те, что ты
присылал в Telegram). Скачай их со своего устройства и переименуй.

## Манифест

### 01 — Гигантские огурцы и сборщики
**Зачем:** главный сюрный донор. Миядзаки-эффект — нелогичный масштаб,
серьёзный мир внутри сюра. Плюс ink+watercolor техника.

### 02 — Бременские музыканты у дома
**Зачем:** динамика, кривоватость, характер. Экспрессивная тушь,
акварельная заливка, движение. Тот самый «прикольный» вайб.

### 03 — Башня с акварелью
**Зачем:** чистый эталон техники. Тонкая тушь, светлая акварель, ясная
композиция. Модель отсюда выучит «вот так выглядит правильный контур».

### 04 — Повозка у кривого дерева
**Зачем:** атмосферный плотный мир с деталями. Каждый уголок что-то
происходит, тёплое освещение, ink+watercolor в насыщенной версии.

### 05 — Зайчиха в дупле дерева (Beatrix Potter-стиль)
**Зачем:** нежность, уют, антропоморфный персонаж, целый интерьер.
Видимая акварель и тончайшие линии.

## Что НЕ кидать сюда

- Тыквы (#4 из присланного) — манга-контур, плоский цвет, утянет в флэт.
- Трое с люстрой (#7) — cell-анимация, не наш медиум.
- Все Ghibli фургоны/замки/караваны (#1, #2, #3, #9, #11) — у них нет
  ink contour, чистая цифровая живопись. Если добавить — модель не
  поймёт, нужен ей контур или нет.
- Lumberjanes (#8) — современный западный комикс, плоско.
- Изумрудный город (#14, #15) — советская классика 70-х, чуть устарела
  для нашего тона.

Миядзаковский сюр и плотность мира мы вытащим **через текстовый промпт**,
который пойдёт вместе с натренированным `style_id`. Стиль (контур+акварель)
даст тренировка, магия мира — текст.

## Что дальше

Когда положишь 5 файлов сюда — скажи мне, я подключу Recraft Custom Style
API: одноразовый скрипт «загрузить эти 5 → получить style_id → записать
в `.env`». Потом меняю в `image.py` style с `digital_illustration/hand_drawn`
на наш свежий `style_id` и деплоим.
